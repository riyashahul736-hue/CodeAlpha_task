# Emotion Recognition System

## Project Overview

The Emotion Recognition System is an Artificial Intelligence and Machine Learning project that detects and predicts human emotions from facial expressions. The system analyzes facial features from an image or live camera feed and classifies the emotion using a trained Machine Learning or Deep Learning model.

---

## Features

- User-friendly web interface
- Facial emotion detection
- Image upload support
- Real-time emotion recognition (optional)
- Fast and accurate prediction
- AI-based emotion classification

## Output

The system predicts one of the following emotions:

- Happy 😊
- Sad 😢
- Angry 😠
- Surprise 😲
- Fear 😨
- Neutral 😐
- Disgust 🤢

---

## Machine Learning / Deep Learning Model

- Convolutional Neural Network (CNN)
- TensorFlow / Keras
- OpenCV for face detection

---

## Project Workflow

1. Upload an image or capture a face using a webcam.
2. Detect the face using OpenCV.
3. Preprocess the image.
4. Extract facial features.
5. Predict the emotion using the trained AI model.
6. Display the detected emotion on the screen.
