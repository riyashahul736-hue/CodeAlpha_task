# AI-Based Credit Scoring and Loan Approval Prediction System

## Project Overview

This project is an AI-Based Credit Scoring and Loan Approval Prediction System developed using Python, Flask, HTML, CSS, JavaScript, and Machine Learning.

The system predicts the credit risk level of a customer based on financial details such as age, monthly income, loan amount, and existing debt. It also displays the estimated credit score and loan approval percentage.

---

## Features

- User-friendly web interface
- AI-based credit risk prediction
- Credit Score Display
- Loan Approval Percentage
- Risk Level Prediction (Low Risk / High Risk)
- Machine Learning Model Integration
- Flask Backend
